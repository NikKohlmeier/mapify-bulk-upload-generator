Mapify Bulk Import Generator

v2.0.0

Purpose:

Generate bulk inport .csv with shortcode descriptions, allowing for more accurate and easy updating

Instructions

  1. Open a new spreadsheet
  2. Copy and paste the following rows (including headers) from the Master SST Info sheet (must be in this specific order):
    Name Address City State Zip Model Hours County
  3. Paste the new rows in, making sure they're in the correct order
  4. Format the hours for the shortcode
    5. Example: Mon - Fri: 6:00 AM - 10:00 PM; Saturday: 10:00 AM - 3:00 PM; Sunday: Closed Save and upload the file below
  5. The file will automatically download to your computer's download folder (default)
  6. Locate file and upload into Mapify
    7. MapifyPro -> Batch Upload -> Choose File -> Import (button)
  7. All of the locations should start to list if they were imported successfully or not
    8. Make a note of any that are unsuccessful, because they will need manually geocoded. Or there is something wrong with the address
